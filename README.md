### 📂 Featured Ecosystem
Instead of building many disconnected mini-projects, I focus on building a complete, deeply integrated e-commerce system:

* 🛍️ **[HandmadeShop_API](https://github.com/tamnguyen2005/HandmadeShop_API)**: The core backend engine. Built with **.NET 8, SQL Server, Redis, and SignalR**. Features JWT Authentication, Unit of Work pattern, and Auto-migrations via Docker.
* 📱 **[HandmadeShop_App](https://github.com/tamnguyen2005/HandmadeShop_App)**: The client-side mobile application. Built with **Flutter & Dart**, directly consuming the .NET API with fully typed models and handling multipart/form-data for image uploads.
